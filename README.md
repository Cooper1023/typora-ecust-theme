# 主题名称（Typora Theme Name）

一款简洁、美观，专为本科生/研究生课程论文排版优化的 Typora 主题。

## ✨ 特色 Features

- 支持中英文排版优化
- 适配论文常用格式（标题、正文、公式、表格）
- 适合课程论文、作业报告、项目总结
- 美化引用块、代码块、数学公式显示

## 🚀 使用方法 How to Use

1. 下载本仓库中的 `.css` 文件
2. 打开 Typora → 偏好设置 → 外观 → 主题 → 打开主题文件夹
3. 将 `.css` 文件复制到主题文件夹中
4. 重新打开 Typora，在主题列表中选择本主题
